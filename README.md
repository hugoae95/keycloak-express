Keycloak - express

1. Generate .env file:
NODE_ENV=
APP_PORT=
KC_SERVER_URL=
KC_REALM=
KC_CLIENT_ID=
KC_CLIENT_SECRET=   

2. Check Keycloak docker and install, generate realm and client authentication
https://www.keycloak.org/getting-started/getting-started-docker

3. Run yarn install.

4. Run yarn start.
